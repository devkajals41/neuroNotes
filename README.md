## вљ™пёЏ Installation

### Clone Repository

```bash
git clone https://github.com/devkajals41/neuroNotes.git
```

### Backend Setup

```bash
cd backend
npm install
```

Create a `.env` file:

```env
PORT=5000
MONGO_URI=YOUR_MONGO_URI
JWT_SECRET=YOUR_SECRET
GEMINI_API_KEY=YOUR_API_KEY
```

Run Backend:

```bash
npm run dev
```

### Frontend Setup

```bash
cd frontend
npm install
npm run dev
```
